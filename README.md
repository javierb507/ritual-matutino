# Ritual Matutino

A bilingual (Spanish/English) Progressive Web App for guided morning and night mindfulness rituals.

## Features

- **Morning Ritual (7 min)**: Neural reprogramming with breathing, visualization, and intention setting
- **Night Ritual (4 min)**: Mental cleanup, goal focus, relaxing breath, and anticipated gratitude
- **Bilingual Support**: Full Spanish and English translations
- **PWA Ready**: Installable on Android as a native-like app
- **Streak Tracking**: Track your daily practice
- **Mood Logging**: Record how you feel after each session
- **Diary History**: Review your past sessions

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion for animations
- Express.js backend
- PostgreSQL with Drizzle ORM

## Installation

```bash
npm install
npm run dev
```
